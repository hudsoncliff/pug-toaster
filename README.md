# pug-toaster

pug boilerplate.
gulp tasks set to compile scss to css, minify javascript files, optimize images in to directory for deployment.Default is set to /assets.

## Directory Structure

```
├─ src
│  |─ css
|  ├─ js
│  └─ img
|  └─ views
|     └─ index.pug
├─gulpfile.js
├─package.json
├─node_modules
├─.gitignore
```



## Package Manager

```
yarn
```
