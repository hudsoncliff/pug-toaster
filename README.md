# pug-toaster

pug boilerplate.
gulp tasks set to compile scss to css, minify javascript files, optimize images in to directory for deployment.Default is set to /dest.

## Directory Structure

```
├─ src				# コンパイルする前のファイルはここで管理
│  |─ assets
│  |  ├─ css
|  |  ├─ js
│  |  └─ scss
|  ├─ includes
|  |  ├─ _foot.pug
│  |  └─ _head.pug
|  └─ index.pug
├─gulpfile.js
├─package.json
├─node_modules
├─.gitignore
```



## Package Manager

```
yarn
```
